java script is used with HTML
to change the content of HTML , 
to change the attribute ,
to change content of CSS , 
to hide or show an element
all these are done at the run time
java script is written inside the body or head of HTML 
syntax 
<body>
<script>
</script>
</body>
it can also be written externally
to access a particular element by its ID
document.getElementById("name of id").innerHTML="new content";
eg:
  <body>
  <p id="myname"></p>
  <script>
  document.getElementById("myname").innerHTML="bijender singh rawat";
  </script>
  </body>                                                                      // by this code "bijender singh rawat" will print
  this java script code could be written inside head tag
but the problem we'll face is that document.getElementById will not get any element's id because it is not there
so to write this type of JS code in head tag , we have to create a function
<head>
<script>
function display() {
document.getElementById("myname").innerHTML="bijender singh rawat";
}
</script>
</head>

validation in JScript
for eg we have a form
<form name="myform">
<input type="text" id="t1">
<button onclick="myvalid()">click me</button>

if we want to make sure that value must be entered in the input box then inside the function myvalid we will write
function myvalid() {
var x = document.getElementById("t1").value;
if(x == "")
alert("enter some value");
else
alert("thanku for the value");
}

if we want to make sure that the value entered by the user must be a number then we will write
function myvalid() {
var x = document.getElementById("t1").value;
if(isNaN(x))
alert("enter numeric value only");
else
alert("thanku for the value");
}

is we want to have a fixed length of input then we will write
function myvalid() {
var x = document.getElementById("t1").value;
if(x.length)
alert("enter a fixed length");
else
alert("thanku for the value");
}

if we want to have a character value in the input then we can write
function myvalid() {
var x = document.getElementById("t1").value;
var r=/^[A-Za-z]+$/;
if(x.match(r))
alert("value ok");
else 
alert("value not ok");
}
